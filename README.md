🌸 Note Taking App 🌸

A simple, interactive, and browser-based note taking application created using **HTML, CSS, and JavaScript**.

📌 Project Overview

This project is a **Note Taking App** where users can easily:
- Add new notes with a title and content  
- Edit notes with **auto-save** functionality  
- Save individual notes using the **per-note save button**  
- Delete notes instantly  
- Export all notes as a **JSON file**  
- Import previously saved notes to continue working  

The app also provides **color-coded notes** for better organization and a **responsive design** that works well on all devices.

🚀 Features
- Add, edit, and delete notes  
- Auto-save while typing  
- Per-note save option  
- JSON export/import for backup  
- Color-coded notes for better visibility  
- Fully responsive and clean user interface  

🛠 Technologies Used
- HTML5  
- CSS3  
- JavaScript  
- LocalStorage for data persistence  
- Font Awesome for icons
